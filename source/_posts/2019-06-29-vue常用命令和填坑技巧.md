* 阻止事件冒泡
  @click.stop

* 阻止默认事件
  @contextmenu.prevent 

* 模板:
```html
{{msg}}		数据更新模板变化
{{*msg}}	数据只绑定一次
{{{msg}}}	HTML转意输出
```

* 用户会看到花括号标记:
```html
v-cloak		防止闪烁, 比较大段落
<span>{{msg}}</span>		->   v-text
{{{msg}}}			->   v-html
```

* Vue.nextTick( [callback, context] )
```js
// 修改数据
vm.msg = 'Hello'
// DOM 还没有更新
Vue.nextTick(function () {
  // DOM 更新了
})
```
